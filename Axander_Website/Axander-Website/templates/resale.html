KMeans Machine Learning Model

KMeans Background Information:

The objective of K-means is simple: group similar data points together and discover underlying patterns.

To achieve this objective, K-means looks for a fixed number (k) of clusters in a dataset.

KMeans is in the family of Unsupervised Learning where we have no variable to predict tied to the data. Instead of having an output, the data only has an input which would be multiple variables that describe the data.
