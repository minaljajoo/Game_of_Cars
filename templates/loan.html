Decision Tree Fundementals

Creating the Tree

  1. Place best attribute of dataset at root of tree
  2. Compare the values of the root attribute with record's attribute
  3. Follow the branch corresponding to that value and jump to next node
  4. Continue comparing record's attribute to other internal nodes of the tree until a leaf node is reached

Key Terms:

  • Information Gain/Geni - try to estimate the information contained by each attribute. The values of each attribute is calculated then ordered by attribute with high value. Information gain assumes categorical. Gini index is assumed continuous. Gini Index is a metric to measure how often a randomly chosen element would be incorrectly identified. It means an attribute with lower gini index should be preferred.

  • Entropy - measures the randomness or uncertainity of a random variable X. By calculating entropy measure of each attribute we can calculate thei information gain

Decision Tree Machine Learning Model

Decision Tree Information:

Decision Trees (DTs) are a non-parametric supervised learning method used for classification and regression. The goal is to create a model that predicts the value of a target variable by learning simple decision rules inferred from the data features.

DecisionTreeClassifier is a class capable of performing multi-class classification on a dataset.
