Machine Learning through TensorFlow


TensorFlow is a framework created by Google Brain Team for creating Deep Learning models. Deep Learning is a category of machine learning models that use multi-layer neural networks. Everything in TensorFlow is based on creating a computational graph, which is a network of nodes, with each node known as an operation, running some function that can be as simple as addition or subtraction to as complex as some multi variate equation. Each operation can be handed a constant, array, matrix or n-dimensional matrix, also known as a tensor. There are input nodes and output nodes, which abstract the entire process to what is inputted into the model and what the outputs of the model are.

Implementation of TensorFlow

Create the Tensorflow Model

The original data was divided into two groups: training and testing. Training contained 80% of the data while testing contained the other 20%.

The next step: Define the important parameters and variable to work with the tensors. Also Define the number of hidden layers and number of neurons for each layer

The training data was fed into the neural network using 600 epochs. The model contained four hidden layers with multiple neurons per layer.

Training was completed with the training data.

The next step: Stored in the Models folder and also plot the MSE and Accuracy for visualization.

The next step: The model was then tested with the test data

Resulted in an accuracy of 1.0. and Mean Square Error as 101.1188

Restore the Tensorflow Model

Once the model was tested and proven to be accurate, the customer data, also known as validation data, can be fed into the model, which was not seen before and a prediction can be obtained.

The First step: Restore the model from the Model folder.

The Next step: Fed the validation data to the model and check for accuracy.

At the moment, the customer data is not defined by user input, but in the future, it will be.
